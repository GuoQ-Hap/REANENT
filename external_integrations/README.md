# 外部对接

本目录只存放外部系统对接相关内容，用于沉淀 PMC 库存供应链智能体与第三方平台、协作工具、业务系统之间的接入方案、接口边界、配置项和验收标准。

## 收录范围

- 外部平台接入方案，例如飞书、企业微信、钉钉、邮件、外部审批系统。
- 外部系统触发智能体的入口设计，例如机器人消息、Webhook、回调事件、定时任务。
- 外部系统权限、密钥、回调地址、白名单、审计与安全要求。
- 外部系统消息格式、字段映射、错误处理、验收清单。

## 不收录范围

- PMC 业务规则本身。
- 内部数据仓库、ERP、MES、WMS、Excel 规则表的数据口径说明。
- Agent 编排、模型路由、工具实现、前端页面实现。
- 运行日志、测试数据、密钥或任何真实敏感配置。
